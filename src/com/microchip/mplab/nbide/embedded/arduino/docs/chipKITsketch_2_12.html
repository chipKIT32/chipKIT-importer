Editing and Rebuilding the Core
When working with the sketch, you may want to make changes to the chipKIT core files that are copied/referenced in the MPLAB X project. The core library file, libCore.a, is not rebuilt automatically when files are changed. It must be rebuilt manually, as follows:
Save the changes, then start the core build by right-clicking on the project in the Projects tab, and select “Rebuild Core Library,” (see Figure: Rebuild Core Library). If there are any issues with building the core due to your changes, they must be resolved in order for the application to be built.
Figure: Rebuild Core Library
You will see the result of the build in the Output window, as shown in Figure 2-xx.
Figure: Output Message
After the core library file is rebuilt successfully, return to the Program the Sketch or Debug the Sketch step to continue your work.
