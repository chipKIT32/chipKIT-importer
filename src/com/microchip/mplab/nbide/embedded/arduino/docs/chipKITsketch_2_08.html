Import the Project
A message displays stating that it is importing the project and it may take a while (see Figure: Importing the Project). During this time, the core libraries are building the program and setting up the project in MPLAB X.
When this process is finished, a message displays and the window closes.
Figure: Importing the Project
The project view has been populated with the source files (.cpp/.c/.S/.h) that are produced by the Arduino IDE when it compiles the Sketch. All the files needed to compile the sketch are added to the project. See Figure: Project View.
