Debug the Sketch
General Information
Note:
On certain boards, the chipKIT bootloader does not support debugging if the bootloader is in the Loadables folder of the sketch. To accommodate those boards, the importer will create a separate configuration from the default configuration called "debug." Select this configuration before attempting to debug the sketch.
The following boards are affected:
• chipKIT DP32
• chipKIT Lenny
• chipKIT Pi
• Fubarino Mini
The main benefit of importing the sketch into MPLAB X IDE is to be able to debug the sketch. You can set breakpoints to stop the program at selected lines to view the code to check for errors and correct them.
