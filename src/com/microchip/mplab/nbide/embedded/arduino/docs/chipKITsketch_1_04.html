Install and Launch the Arduino IDE
Download the Arduino IDE (version 1.8.1 minimum) from:
https://www.arduino.cc/en/Main/Software.
For Windows users, download arduino-1.8.1-windows (or newer) and run the installer. Refer to the Arduino installation instructions on their web site.
Launch Arduino IDE from the Start menu on your computer.
