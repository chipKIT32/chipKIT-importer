Select the Tool
MPLAB X IDE displays Hardware Tools so you can select one as a debugger. If you have any hardware tools connected (PICkit 3, MPLAB ICD 3, etc.), their serial numbers display in the list. The Simulator is built into MPLAB X IDE so no connection is required and no serial number displays. For this example, however, select the serial number (SN:) associated with the MPLAB ICD 3. See Figure: Select Hardware Tool.
