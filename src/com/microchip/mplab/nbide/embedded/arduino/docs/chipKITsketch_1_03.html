Before You Begin
There are many different chipKit-supported boards, PIC32-supported development tools (MPLAB ICD 3, PICkit 3, MPLAB REAL ICE, etc.) that you can use to debug a sketch, however, this document presents an example using the following:
•
Arduino IDE, v1.8.1 or later
•
MPLAB X IDE, v3.55 or later
•
MPLAB XC32 C Compiler, v1.43 or later
•
chipKIT Max32 board (TDGL003)
•
MPLAB ICD 3 In-Circuit Debugger (DV164035)
•
USB cable
•
6-inch modular cable (07-00024)
•
SIL Adapter: 02-01988 ASSY, RJ11 Jack to SIL Adapter, SIL connector cable (AC164110)
•
9V power supply (AC002014)
 
In this section, you are guided through the following steps:
•
Install and Launch the Arduino IDE
•
Add the chipKIT Core Repository to the Boards Manager in Arduino
•
Install the Core in Arduino
•
Select the Board in Arduino
•
Select the COM Port in Arduino
