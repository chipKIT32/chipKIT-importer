Install the Core in Arduino
Next, select Tools>Board>Boards Manager to open the Boards Manager dialog (see Figure: Open the Boards Manager).
Figure: Open the Boards Manager
When the Boards Manager dialog opens, type "chipKIT" in the search box to find the "chipKIT by chipKIT Community." Click anywhere in the body of the text that displays to reveal the Install button. Click Install (see Figure: Install the Core). After the core is installed, close the Boards Manager window.
