Program the Sketch
With the chipKIT Sketch Importer you can program the target board similar to the Arduino IDE. The benefits of using the plugin with MPLAB X IDE include faster compile times and the core libraries do not have to be recompiled every time the sketch is rebuilt.
The plugin adds two items to the toolbar, as shown below:
Figure: Toolbar ICons
Select COM Port
Attach the board to the computer using a USB cable. Then select the COM port for the board by clicking the new drop-down box.
Program the Board
Click the "cK" button next to the drop-down box, and the project will be compiled (if necessary), and the resulting hex file will be programmed through the selected COM port.
Some chipKIT boards require a special step to be placed in programming mode, such as holding a PGM button while pressing and releasing the RESET button on the board. Make sure that the step is executed just before or just after clicking the programming button.
