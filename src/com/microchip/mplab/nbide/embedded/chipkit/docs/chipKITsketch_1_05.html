Add the chipKIT Core Repository to the Boards Manager in Arduino
To create a sketch that will run on chipKIT boards, you need to add the chipKIT Core repository for the Arduino IDE. Select File>Preferences (see Figure: Edit Preferences in Arduino IDE) from the Arduino IDE to open the Preferences dialog.
Figure: Edit Preferences in Arduino IDE
In the Preferences dialog (Figure: Arduino Preferences Dialog), type to the following URL in the "Additional Boards Manager URLs" field:
https://github.com/chipKIT32/chipKIT-core/raw/master/
package_chipkit_index.json
Then, click OK.
Figure: Arduino Preferences Dialog
