Set Breakpoint
You can set breakpoints in the project files (.ino for sketch files, .ccp for C++ files, etc.). For our example, youâ€™ll set a breakpoint in the .ino file.
Open blink.ino and locate the line of code digitalWrite (PIN_LED1, HIGH). Set a breakpoint (click on the line number or right click on the text and select Toggle Line Breakpoint). See Figure: Set Breakpoint. Once the breakpoint is set, the line of code will be highlighted in red so you can quickly see where the breakpoint is located.
Figure: Set Breakpoint
 
Then select Debug>Debug Project or click the Debug Project icon .
 
Notice the highlighting changes to green and you can now step in the .ino file (see Figure: Step in).
Figure: Step in
In addition to setting breakpoints, you can watch variables, etc. Refer to the online help for the MPLAB X IDE (Help>Tool Help Contents>MPLAB X IDE Help) for information on these and other debugging topics.
