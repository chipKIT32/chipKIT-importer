Introduction
Microchip developed the chipKIT Sketch Importer plug-in enables you to import a sketch created in Arduino IDE (Integrated Development Environment) into the Microchip MPLAB X IDE. Once the sketch is imported, you can debug your sketch (code).
This document provides an example for creating a sketch in Arduino IDE, installing the plug-in, importing the sketch into MPLAB X IDE, and debugging the sketch.
