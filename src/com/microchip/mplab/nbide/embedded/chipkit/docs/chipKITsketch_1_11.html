Test the Sketch
Test the sketch to verify it builds and runs on the target board before importing it into MPLAB X IDE. To do this, select Sketch>Upload, or click the Upload icon on the main screen (see Figure: Arduino Toolbar for toolbar icons).
Figure: Arduino Toolbar
When successful, the sketch is compiled and uploaded to the board. The output window displays a message that the sketch has been uploaded.
You are now ready to import the sketch into the MPLAB X IDE for debugging. Refer to“Import Sketch” for instructions on importing the sketch.
