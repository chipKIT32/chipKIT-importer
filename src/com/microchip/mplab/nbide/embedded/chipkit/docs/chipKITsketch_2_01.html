Introduction
In the previous section, you created a sketch in Arduino IDE. In this section, you’ll install the plug-in, set up the board and the development tool, and import the sketch into MPLAB X IDE. Then, you can program or debug the sketch.
In MPLAB X IDE, when you install the chipKIT Sketch Importer plug-in, an application wizard will import a sketch and convert it into an MPLAB X project.
